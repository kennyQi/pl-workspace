中智票量简介

中智国际旅行社有限公司简称中智票量，前身是中智航空运输服务有限公司，成立于1998年。2008年经国家工商总局和国家旅游局批准，成为具有独立企业法人的具有出境旅游资质的国际旅行社，是中智集团下属全资子公司，注册资金2000万

发轫于1998年的中智航空运输服务有限公司，历经十年的大浪淘沙，在新世纪里呈现出蓬勃发展的生机，期待着辉煌的未来。作为国内航空运输服务业的先行者，中智航运一直发挥着行业领跑的作用，为政府部门、国内企事业单位和国际知名企业在京办事机构提供专业化的服务。

2008年，伴随着公司业务不断走向成熟，在中智总公司领导的大力支持下，中智航运大踏步的推进公司扩展，推崇革新，组建专业精英团队，打造一流的国际旅行服务品牌。

中智国际旅行社发展历程：

1998年，中智航空运输服务有限公司成立。

2008年，更名为中智国际旅行社有限公司。
